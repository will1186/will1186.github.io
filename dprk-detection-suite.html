DPRK Threat Actor Detection Suite

Detection Engineering · 2026
YARA Sigma Python MITRE ATT&CK DPRK

Detection rules and an automated testing framework targeting DPRK-nexus cyber operations — Lazarus Group (HIDDEN COBRA), APT38, and APT45 (Andariel). Built to demonstrate end-to-end detection engineering: from threat intelligence reporting to deployable detection logic with measurable performance metrics.

What It Does

Nine YARA rules identify DPRK malware artifacts in binaries — credential harvesting toolkits, cryptocurrency theft modules, clipboard hijackers, and supply chain backdoors. Six Sigma rules detect behavioral patterns in log telemetry — lateral movement via RDP tunneling, crypto wallet exfiltration chains, and IT worker fraud indicators including RAT stacking and source code theft.

A Python testing framework validates every rule against simulated cloud audit logs containing a mix of malicious, benign, and adversarial false-positive-test events. The scorer reports true positive, false positive, and false negative rates per rule with precision, recall, and F1 metrics.
